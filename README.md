# Text To Sign Language Converter

## About

- This tool is meant to convert text to American Sign Language (ASL).
- It is important for people who are learning ASL to practice their skills as well as learn new phrases and words.
- Or, if you're just interested in ASL, this is the place for you!

## Setup

1. Open the terminal on your IDE.
2. From the terminal, run the following command: python (or python3) manage.py runserver.
3. The terminal will show the localhost address: server at http://127.0.0.1:8000/". This can be run on a browser.
4. Type in the text box to convert your text to ASL.
5. Build your streak reflex game, where you are tested with questions to practice your ASL.
