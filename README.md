# Text To Sign Language Converter

## Setup

1. Open the terminal on your IDE.
2. From the terminal, run the following command: python (or python3) manage.py runserver.
3. The terminal will show the localhost address: server at http://127.0.0.1:8000/". This can be run on a browser.
4. Type in the text box to convert your text to American Sign Language (ASL)
